# CapstoneDesign

## 채팅 구현 
### Server 세팅
-mkdir server

-npm init을 통해 package.json 파일 생성 

-npm install express

-npm install socket.io

### 클라이언트

-npm i query-string

-npm i socket.io-client
클라이언트 에서 socket 설치

-npm i react-emoji
이모지를 사용하기위해 설치

-npm i react-scroll-to-bottom
버튼을 누르면 최신으로 이동하기 위해 설치
